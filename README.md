# Growth-Skill

> 通过日常观察，追踪任何人的认知成长轨迹。

nuwa-skill 蒸馏了名人**怎么想**。growth-skill 追踪一个人**怎么变**。

---

## 核心区别

| 维度 | nuwa-skill | growth-skill |
|------|-----------|--------------|
| 数据来源 | 公开网络搜索 | 个人日常观察 |
| 时间性 | 一次性蒸馏 | 持续纵向积累 |
| 输出 | 静态 Skill | 动态演化的认知画像 |
| 对象 | 名人 | 任何人（宝宝、老人、朋友、自己） |
| 验证方式 | 固定三重验证 | 自适应阈值（数据少时宽松，多时严格） |
| 矛盾处理 | 保留为张力 | 时间矛盾 = 成长核心证据 |
| 记忆机制 | 无 | 模拟人脑记忆周期（遗忘曲线、再巩固、图式沉淀） |

---

## 安装

```bash
npx skills add .
```

## 使用

### 开始追踪

```
> 追踪 Leo，我的儿子，深度追踪
> 追踪 奶奶
```

### 记录观察

```
> 今天Leo第一次自己站起来了
> 记录：他碰到烫的东西缩回手，但第二天又去摸
> 奶奶今天说年轻人太急了，什么都想要快的
```

直接说观察到的事情就行，自由文本，不需要任何格式。

### 查看成长

```
> Leo的成长报告
> 复盘奶奶
> 详细复盘
```

### 手动演化

```
> 演化 Leo
> 更新画像
```

### 管理记录

```
> 删除记录 2026-01-05
> 修正 2026-03-10
```

### 追踪总览

```
> 看看状态
```

## 数据结构

```
data/
  registry.json                 # 所有追踪对象索引
  [person-id]/
    person.json                 # 元信息
    observations/               # 原始日记
    signals.jsonl               # 工作记忆（活跃信号）
    signals-archive/            # 归档信号（模拟人脑记忆层级）
      consolidating/            # 巩固区（按月）
      long-term/                # 长期记忆（按年）
      schemas/                  # 图式记忆（永久沉淀）
    profile.md                  # 当前认知画像
    profile-history/            # 画像历史快照
```

---

## 记忆周期

信号归档模拟人脑记忆的四个阶段：

1. **工作记忆**（signals.jsonl）— 容量有限，保留全部细节
2. **巩固区**（consolidating/）— 按月归档，聚类合并，提取模式摘要
3. **长期记忆**（long-term/）— 按年归档，只保留信号骨架
4. **图式记忆**（schemas/）— 已确立模式的永久沉淀，不遗忘

遗忘遵循 Ebbinghaus 曲线：`retention = e^(-t / stability)`，不同信号类型遗忘速度不同。重复引用会增强 stability（间隔重复效应）。被遗忘的信号被新观察匹配时会再巩固。

## 认知画像包含什么

- **核心心智模型** — 此人用什么框架看世界
- **决策启发式** — 此人怎么做判断
- **表达模式** — 此人的认知框架偏好
- **价值观与反模式** — 此人追求什么、拒绝什么
- **成长时间线** — 认知变化的关键节点
- **矛盾与张力** — 成长的核心证据
- **观察盲区** — 哪些维度缺乏数据
- **观察者偏差** — 系统性的记录倾向
- **诚实边界** — 画像做不到什么
